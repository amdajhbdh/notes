## Exercice 1: Estérification (4 pts)

### Données

### Questions

- Nommer les produits (1 pt)

**2.** Expérience: $n_{acide} = n_{alcool} = 0,2$ mol
- 10 ampoules à $100°C$
- Dosage par $NaOH$: $C_b = 1$ mol/L

**2-1.** Pourquoi refroidir l'ampoule? (0,5 pt)

**2-2.** 

**2-2-1.** Relation entre $n_A$ et $n_E$
- Expression de $n_E$ en fonction de $V_b$ (1 pt)

**2-2-2.** Compléter le tableau:

| $t$ (min) | 0 | 3 | 8 | 28 | 38 | 48 | 68 |
|-----------|---|----|----|----|----|----|-----|
| $V_b$ (cm³) | 20 | 16 | 13,5 | 8,5 | 7,2 | 6,9 | 6,9 |
| $n_E$ (mol) | | | | | | | |

- Tracer $n_E = f(t)$
- Vitesse à $t = 12$ min
