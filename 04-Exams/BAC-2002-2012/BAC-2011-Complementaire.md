## Exercice 1: Hydrolyse (4 pts)

### Données

Ester $E$: $C_6H_{12}O_2$ + $H_2O$ → $A$ + $B$

### Tests

| Composé | Test $KMnO_4$/H$^+$ | Résultat |
|---------|----------------------|----------|
| A | | Violacé (pas de réaction) |
| B | | Décoloration → C |
| C | DNPH | Précipité jaune |
| C | Fehling | Pas de réaction |

### Questions

**1.** 
- Fonctions de $A$, $B$, $C$?
- Justifier (1 pt)

**2.** 
- $m_A = 3$ g
- Solution acide
- $V_{NaOH} = 100$ mL, $C = 0,5$ mol/L
- Déterminer la masse molaire de $A$ (1 pt)

**3.** 
- Formule de $A$, $B$, $E$ (1 pt)
