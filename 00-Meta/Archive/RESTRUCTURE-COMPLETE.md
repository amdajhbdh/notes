## Technical Details

### Database Schema

**fts_notes** (Full-text search)
- path, title, content, tags, subject

**notes** (Metadata)
- path, title, subject, type, difficulty, status
- created, modified, word_count, link_count

**links** (Graph)
- source_path, target_path, link_type

**tags** (Taxonomy)
- tag, count

**formulas** (LaTeX)
- path, formula, context

**problems** (Practice tracking)
- path, problem_num, difficulty, solved, topic
