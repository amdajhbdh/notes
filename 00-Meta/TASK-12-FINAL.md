## 🚀 Usage

> [!example] How to Use
> 
> ### Enable Styling
> ```bash
> # In Obsidian:
> Settings → Appearance → CSS snippets → Enable "vault-styling"
> ```
> 
> ### Enhance Notes
> ```bash
> # Single note
> python3 ../content-enhancer.py enhance "path/to/note.md"
> 
> # All concept notes
> python3 ../content-enhancer.py enhance-all 01-Concepts
> 
> # All textbook notes
> python3 ../content-enhancer.py enhance-all 05-Extracted/Resources
> ```
>
