# Complex Numbers - Basics

← Back to [[Complex Numbers MOC]]

## Definition

A **complex number** is a number of the form:

$$z = a + bi$$

Where:
- $a$ = real part, denoted $\text{Re}(z)$
- $b$ = imaginary part, denoted $\text{Im}(z)$
- $i$ = imaginary unit, where $i**2 = -1$

## Key Concepts

### The Imaginary Unit
$$i = \sqrt{-1}$$
$$i**2 = -1$$
$$i**3 = -i$$
$$i**4 = 1$$

Pattern repeats every 4 powers!

### Real vs Imaginary
- If $b = 0$: $z = a$ (purely real)
- If $a = 0$: $z = bi$ (purely imaginary)
- If $a, b \neq 0$: $z$ is a complex number

### Equality
Two complex numbers are equal if and only if:
$$a + bi = c + di \iff a = c \text{ and } b = d$$

## The Complex Plane (Argand Diagram)

![[complex-plane-diagram.excalidraw]]

- **Horizontal axis**: Real part
- **Vertical axis**: Imaginary part
- Each complex number is a point $(a, b)$

## Examples

### Example 1: Identify Parts
For $z = 3 + 4i$:
- $\text{Re}(z) = 3$
- $\text{Im}(z) = 4$

### Example 2: Powers of i
Calculate $i**{17}$:
$$i**{17} = i**{16} \cdot i = (i**4)**4 \cdot i = 1**4 \cdot i = i$$

### Example 3: Equality
If $2x + 3yi = 6 + 9i$, find $x$ and $y$:
- $2x = 6 \implies x = 3$
- $3y = 9 \implies y = 3$

2. Calculate $i**{25}$
