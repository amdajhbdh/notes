## Mathematical Formulas
- **GCD**: $\gcd(a,b) = \text{largest } d \text{ such that } d|a \text{ and } d|b$
- **LCM**: $\text{lcm}(a,b) = \text{smallest } m \text{ such that } a|m \text{ and } b|m$
