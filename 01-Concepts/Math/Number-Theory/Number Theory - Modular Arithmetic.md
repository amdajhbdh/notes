# Number Theory - Modular Arithmetic

> Arithmetic modulo n

## Key Concepts
1. Congruence
2. Modular Addition
3. Modular Multiplication
4. Modular Exponentiation
5. Applications

## Mathematical Formulas
- **Congruence**: $a \equiv b \pmod{n}$ means $n | (a-b)$
- **Modular Inverse**: $a \cdot a^{-1} \equiv 1 \pmod{n}$
