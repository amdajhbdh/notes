# Kinetics - Rate Laws

> Mathematical relationship between rate and concentration

## Key Concepts
1. Rate Expression
2. Rate Constants
3. Order of Reaction
4. Integrated Rate Laws
5. Half-life

## Mathematical Formulas
- **Rate Law**: $Rate = k[A]^m[B]^n$
- **First Order**: $\ln[A] = \ln[A_0] - kt$
- **Second Order**: $\frac{1}{[A]} = \frac{1}{[A_0]} + kt$
- **Half-life (1st order)**: $t_{1/2} = \frac{\ln 2}{k}$
