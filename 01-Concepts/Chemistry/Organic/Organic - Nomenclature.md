# Organic - Nomenclature

> IUPAC naming conventions for organic compounds

## Key Concepts
1. IUPAC Rules
2. Root Names (meth-, eth-, prop-, but-)
3. Functional Group Prefixes/Suffixes
4. Numbering Rules
5. Systematic Naming

## Functional Group Names
| Group | Prefix | Suffix |
|-------|--------|--------|
| Alkene | -ene | -ene |
| Alkyne | -yne | -yne |
| Alcohol | hydroxy- | -ol |
| Aldehyde | oxo- | -al |
| Ketone | oxo- | -one |
| Carboxylic acid | carboxy- | -oic acid |
| Amine | amino- | -amine |

## Example
2-methylpropanoic acid:
$$CH_3-CH(CH_3)-COOH$$
