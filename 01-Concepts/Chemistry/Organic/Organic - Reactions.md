# Organic - Reactions

> Common organic reaction types

## Key Concepts
1. Addition Reactions
2. Elimination Reactions
3. Substitution Reactions
4. Oxidation-Reduction
5. Polymerization

## Reaction Types

### Addition
- Halogenation: $R-CH=CH_2 + X_2 \rightarrow R-CHX-CH_2X$
- Hydration: $R-CH=CH_2 + H_2O \rightarrow R-CHOH-CH_3$

### Elimination
- Dehydration: $R-CH_2-CH_2OH \rightarrow R-CH=CH_2 + H_2O$

### Substitution
- Nucleophilic: $R-X + Nu^- \rightarrow R-Nu + X^-$
- Free radical: $CH_4 + Cl_2 \rightarrow CH_3Cl + HCl$
