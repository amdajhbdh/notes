# Hydrocarbons

## Definition
Organic compounds containing only carbon and hydrogen atoms.

## Classification

### 1. Alkanes (Saturated)
Single bonds only (C-C)

**General Formula**: C_nH_2_n₊_2

**Examples**:
- Methane: CH_4
- Ethane: C_2H_6
- Propane: C_3H_8
- Butane: C_4H_1_0

**Properties**:
- Least reactive
- Undergo substitution reactions
- Combustion: C_nH_2_n₊_2 + O_2  ->  CO_2 + H_2O

### 2. Alkenes (Unsaturated)
One or more double bonds (C=C)

**General Formula**: C_nH_2_n

**Examples**:
- Ethene: C_2H_4
- Propene: C_3H_6
- Butene: C_4H_8

**Properties**:
- More reactive than alkanes
- Undergo addition reactions
- Test: Decolorize bromine water

### 3. Alkynes (Unsaturated)
One or more triple bonds (C≡C)

**General Formula**: C_nH_2_n₋_2

**Examples**:
- Ethyne (Acetylene): C_2H_2
- Propyne: C_3H_4

**Properties**:
- Most reactive
- Undergo addition reactions
- Used in welding torches

### 4. Aromatic Hydrocarbons
Contain benzene ring (C_6H_6)

**Structure**: Hexagonal ring with alternating double bonds

**Examples**:
- Benzene: C_6H_6
- Toluene: C_7H_8
- Naphthalene: C_1_0H_8

## Isomers
Compounds with same molecular formula but different structures

**Example**: C_4H_1_0
- n-butane (straight chain)
- isobutane (branched)

## Reactions

### Combustion
Complete: C_nH_m + O_2  ->  CO_2 + H_2O
Incomplete: C_nH_m + O_2  ->  CO + H_2O (or C + H_2O)

### Substitution (Alkanes)
CH_4 + Cl_2  ->  CH_3Cl + HCl (with UV light)

### Addition (Alkenes)
C_2H_4 + H_2  ->  C_2H_6 (hydrogenation)
C_2H_4 + Br_2  ->  C_2H_4Br_2
