# Système Nerveux

## Organisation
- **SNC:** Cerveau + Moelle épinière
- **SNP:** Nerfs périphériques

## Neurone
- Dendrites → Corps cellulaire → Axone
- Transmission: Électrique + Chimique

## Synapse
1. Potentiel d'action
2. Libération neurotransmetteurs
3. Récepteurs postsynaptiques

## Réflexe
Arc réflexe: Récepteur → Neurone sensitif → Moelle → Neurone moteur → Effecteur
