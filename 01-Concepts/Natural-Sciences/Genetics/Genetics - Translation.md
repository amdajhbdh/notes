# Genetics - Translation

> RNA to protein synthesis

## Key Concepts
1. Genetic Code
2. Codon Table
3. Ribosome Structure
4. Initiation Complex
5. Elongation Cycle
6. Termination

## The Genetic Code
- 64 codons (61 encode, 3 stop)
- Start codon: AUG (Methionine)
- Stop codons: UAA, UAG, UGA

## Process Steps

### 1. Initiation
- Ribosome assembles at 5' end of mRNA
- Start codon (AUG) located
- Initiator tRNA binds

### 2. Elongation
- Codon-anticodon pairing
- Peptide bond formation
- Ribosome translocates

### 3. Termination
- Stop codon reached
- Release factor binds
- Polypeptide released
