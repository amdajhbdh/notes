# Nervous - Synapses

> Connection between neurons

## Key Concepts
1. Synapse Structure
2. Neurotransmitters
3. Synaptic Transmission
4. Excitation/Inhibition
5. Synaptic Plasticity

## Synapse Types

### Electrical Synapse
- Direct cytoplasmic connection
- Gap junctions
- Fast, bidirectional

### Chemical Synapse
- Neurotransmitter release
- Slower, unidirectional

## Synaptic Transmission

### 1. Action Potential Arrives
- Voltage-gated Ca2+ channels open

### 2. Vesicle Fusion
- Ca2+ triggers exocytosis
- Neurotransmitter release

### 3. Receptor Binding
- Binds to postsynaptic receptors
- Opens ion channels

### 4. Response
- EPSP (excitatory)
- IPSP (inhibitory)

### 5. Termination
- Reuptake
- Enzymatic degradation

## Neurotransmitters
- **Acetylcholine**: Muscles, learning
- **Dopamine**: Reward, movement
- **Serotonin**: Mood, sleep
- **GABA**: Inhibition
- **Glutamate**: Excitation
