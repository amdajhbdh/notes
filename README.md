# BAC Resources - Study Notes

Organized study materials for Mauritanian Baccalauréat preparation.

## Directory Structure

- **[00-Meta/](00-Meta/)** - System files, archives, and project management
- **[01-Concepts/](01-Concepts/)** - Fundamental concepts by subject
- **[02-Practice/](02-Practice/)** - Practice problems and exercises  
- **[03-Resources/](03-Resources/)** - Reference materials and textbooks
- **[04-Exams/](04-Exams/)** - Past exam papers and solutions
- **[05-Extracted/](05-Extracted/)** - Content extracted from textbooks
- **[Animations/](Animations/)** - Educational animations and visualizations
- **[mauritanian-bac/](mauritanian-bac/)** - Official BAC materials

## Quick Start

1. Start with **[00-READ-ME-FIRST.md](00-READ-ME-FIRST.md)**
2. Browse concepts in **[01-Concepts/](01-Concepts/)**
3. Practice with **[02-Practice/](02-Practice/)**
4. Review past exams in **[04-Exams/](04-Exams/)**

## Subject Areas

Each category is organized by:
- **Math** - Complex numbers, matrices, integrals, number theory
- **Physics** - Dynamics, electromagnetism, circuits, mechanics  
- **Chemistry** - Organic chemistry, solutions, kinetics
- **Biology** - Genetics, nervous system, reproduction
