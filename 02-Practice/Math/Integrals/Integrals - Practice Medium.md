# Integrals - Practice Medium

## Problem 1: Polynomial Integration
∫ (4x**3 - 6x**2 + 2x - 5) dx

<details>
<summary>Solution</summary>

∫ (4x**3 - 6x**2 + 2x - 5) dx
= x**4 - 2x**3 + x**2 - 5x + C
</details>

## Problem 2: Definite Integral
∫[1 to 3] (x**2 + 2x) dx

<details>
<summary>Solution</summary>

F(x) = x**3/3 + x**2

F(3) - F(1) = (27/3 + 9) - (1/3 + 1)
= (9 + 9) - (1/3 + 1)
= 18 - 4/3
= 50/3
</details>

## Problem 4: Trigonometric
∫ (4sin(x) - 3cos(x)) dx

<details>
<summary>Solution</summary>

∫ (4sin(x) - 3cos(x)) dx
= -4cos(x) - 3sin(x) + C
</details>

## Problem 5: Mixed Functions
∫[0 to π/2] (sin(x) + cos(x)) dx

<details>
<summary>Solution</summary>

F(x) = -cos(x) + sin(x)

F(π/2) - F(0) = (-cos(π/2) + sin(π/2)) - (-cos(0) + sin(0))
= (0 + 1) - (-1 + 0)
= 1 + 1
= 2
</details>

## Problem 7: Area Under Curve
Find the area between y = x**2 and the x-axis from x = 0 to x = 2

<details>
<summary>Solution</summary>

Area = ∫[0 to 2] x**2 dx
= [x**3/3] from 0 to 2
= 8/3 - 0
= 8/3 square units
</details>

## Problem 8: Substitution Preview
∫ 2x(x**2 + 1)**5 dx

<details>
<summary>Solution</summary>

Let u = x**2 + 1, du = 2x dx

∫ u**5 du = u**6/6 + C
= (x**2 + 1)**6/6 + C
</details>
