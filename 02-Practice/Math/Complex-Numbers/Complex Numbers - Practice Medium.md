## Problems

### Division (1-5)

**1.** Calculate: $\frac{3 + 4i}{1 - 2i}$

**2.** Calculate: $\frac{2 - i}{3 + i}$

**3.** Simplify: $\frac{1}{i}$

**4.** Calculate: $\frac{5 + 2i}{2 - 3i}$

**5.** Simplify: $\frac{1 + i}{1 - i}$

### Equations (6-10)

**6.** Solve: $z**2 = -1$

**7.** Solve: $z**2 - 2z + 5 = 0$

**8.** Solve: $z**2 + 4 = 0$

**9.** If $|z| = 5$ and $\text{Re}(z) = 3$, find $z$

**10.** Find all $z$ where $z**2 = 3 + 4i$

### Polar Form (11-15)

**11.** Convert $z = 1 + i$ to polar form

**12.** Convert $z = -1 + i\sqrt{3}$ to polar form

**13.** If $z = 2(\cos 60° + i \sin 60°)$, find rectangular form

**14.** Find the argument of $z = -3 - 3i$

**15.** Calculate $(1 + i)**6$ using De Moivre's theorem

### Properties (16-20)

**16.** If $z_1 = 2 + 3i$ and $z_2 = 1 - i$, find $|z_1 \cdot z_2|$

**17.** Prove: $|z|**2 = z \cdot \bar{z}$ for $z = a + bi$

**18.** Find $\text{Re}(z**2)$ if $z = 3 + 2i$

**19.** If $|z - 2| = 3$, describe the locus geometrically

**20.** Calculate: $(2 + i)**3$

## Answers

<details>
<summary>Click to reveal answers</summary>

1. $-1 + 2i$
   - Multiply by conjugate: $\frac{(3+4i)(1+2i)}{(1-2i)(1+2i)} = \frac{3+6i+4i-8}{5} = \frac{-5+10i}{5}$

2. $\frac{1}{2} - \frac{1}{2}i$
   - $\frac{(2-i)(3-i)}{(3+i)(3-i)} = \frac{6-2i-3i-1}{10} = \frac{5-5i}{10}$

3. $-i$
   - $\frac{1}{i} \cdot \frac{-i}{-i} = \frac{-i}{1}$

4. $\frac{4}{13} + \frac{19}{13}i$

5. $i$
   - $\frac{(1+i)**2}{(1-i)(1+i)} = \frac{2i}{2} = i$

6. $z = \pm i$

7. $z = 1 \pm 2i$
   - Using quadratic formula: $z = \frac{2 \pm \sqrt{4-20}}{2} = \frac{2 \pm 4i}{2}$

8. $z = \pm 2i$

9. $z = 3 \pm 4i$
   - Since $|z|**2 = 9 + b**2 = 25$, so $b = \pm 4$

10. $z = 2 + i$ or $z = -2 - i$

11. $z = \sqrt{2}(\cos 45° + i \sin 45°) = \sqrt{2}e**{i\pi/4}$
    - $r = \sqrt{1**2 + 1**2} = \sqrt{2}$, $\theta = 45°$

12. $z = 2(\cos 120° + i \sin 120°)$
    - $r = 2$, $\theta = 120°$

13. $z = 1 + i\sqrt{3}$
    - $2(\frac{1}{2} + i\frac{\sqrt{3}}{2})$

14. $\arg(z) = -135°$ or $\frac{5\pi}{4}$

15. $-8i$
    - $(\sqrt{2})**6 \cdot (\cos 270° + i \sin 270°) = 8(0 - i)$

16. $|z_1 \cdot z_2| = \sqrt{13} \cdot \sqrt{2} = \sqrt{26}$

17. $z \cdot \bar{z} = (a+bi)(a-bi) = a**2 + b**2 = |z|**2$ 

18. $\text{Re}(z**2) = \text{Re}((3+2i)**2) = \text{Re}(9+12i-4) = 5$

19. Circle with center at $(2, 0)$ and radius $3$

20. $2 + 11i$
    - $(2+i)**3 = 8 + 12i + 6i**2 + i**3 = 8 + 12i - 6 - i$
