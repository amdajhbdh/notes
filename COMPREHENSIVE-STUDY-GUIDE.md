## 🔑 Key Formulas by Topic

### Complex Numbers
```
z = a + bi
|z| = √(a² + b²)
z = r·e^(iθ) = r(cos θ + i sin θ)
De Moivre: [r(cos θ + i sin θ)]^n = r^n(cos nθ + i sin nθ)
```

### Integrals
```
∫xⁿ dx = x^(n+1)/(n+1) + C
∫eˣ dx = eˣ + C
∫sin(x) dx = -cos(x) + C
∫cos(x) dx = sin(x) + C
∫u dv = uv - ∫v du (Integration by parts)
```

### Matrices
```
det([[a,b],[c,d]]) = ad - bc
A⁻¹ = (1/det(A)) × adj(A)
(AB)ᵀ = BᵀAᵀ
det(AB) = det(A)·det(B)
```

### Dynamics
```
F = ma (Newton's Second Law)
v = v₀ + at
s = v₀t + ½at²
v² = v₀² + 2as
p = mv (Momentum)
W = F·s·cos(θ) (Work)
```

### Electric Fields
```
E = F/q
F = k(q₁q₂)/r² (Coulomb's Law)
E = kq/r² (Point charge)
V = kq/r (Potential)
U = qV (Potential energy)
```

### Circuits
```
V = IR (Ohm's Law)
P = VI = I²R = V²/R
Series: R_total = R₁ + R₂ + ...
Parallel: 1/R_total = 1/R₁ + 1/R₂ + ...
```

### Organic Chemistry
```
Alkanes: CₙH₂ₙ₊₂
Alkenes: CₙH₂ₙ
Alkynes: CₙH₂ₙ₋₂
Combustion: CₓHᵧ + O₂ → CO₂ + H₂O
```

### Kinetics
```
rate = k[A]ᵐ[B]ⁿ
First order: ln[A] = ln[A]₀ - kt
Second order: 1/[A] = 1/[A]₀ + kt
k = Ae^(-Eₐ/RT) (Arrhenius)
```

### Genetics
```
Monohybrid: 3:1 ratio
Dihybrid: 9:3:3:1 ratio
Punnett square for predictions
Hardy-Weinberg: p² + 2pq + q² = 1
```

## 💡 Problem-Solving Strategies

### For Mathematics Problems:
1. Identify the type of problem
2. List known and unknown variables
3. Select appropriate formula
4. Substitute values carefully
5. Check units and reasonableness

## ⚠️ Common Mistakes to Avoid

### Mathematics
- ❌ Forgetting ±√ when solving quadratic equations
- ❌ Incorrect sign in integration by parts
- ❌ Not checking domain restrictions
- ❌ Arithmetic errors in matrix operations

### Physics
- ❌ Forgetting to convert units
- ❌ Ignoring direction of vectors
- ❌ Incorrect free body diagrams
- ❌ Sign errors in equations

### Chemistry
- ❌ Unbalanced chemical equations
- ❌ Incorrect molar mass calculations
- ❌ Forgetting to account for stoichiometry
- ❌ Rounding too early

### Biology
- ❌ Incorrect Punnett square setup
- ❌ Confusing dominant and recessive
- ❌ Wrong probability calculations
- ❌ Misinterpreting pedigree charts
