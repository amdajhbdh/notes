## 🧬 BIOLOGY EXERCISES

### **Bio_Ex1: Hormone Regulation (Joussour SN - Exercise 1)**
**Problem:** Two women (M & N) with absent menstruation. Analyze hormone levels.

**Data to visualize:**
- Document 1: Ovarian hormone levels (30 days) - both women = 0
- Document 2: FSH/LH levels
  - Woman M: FSH=92, LH=60
  - Woman N: FSH=4, LH=3
  - Normal: FSH=32, LH=30
- Document 3: Ovarian ultrasound comparisons
- Document 4: GnRH injection test results

**Animation Scenes:**
1. **Scene 1a** - Graph showing zero ovarian hormones for M & N vs normal cycle
2. **Scene 1b** - Explain why no menstruation (no hormone  ->  no endometrium)
3. **Scene 2a** - Bar chart comparing FSH/LH: M (high), N (low), Normal (baseline)
4. **Scene 2b** - Two hypotheses for each woman (ovarian failure vs pituitary issue)
5. **Scene 3** - Ovarian structure comparison (follicles present/absent)
6. **Scene 4** - GnRH test response graph  ->  diagnose pituitary vs ovarian cause

**Duration:** ~3 minutes

### **Bio_Ex2: Synaptic Transmission (Joussour SN - Exercise 2)**

**Data to visualize:**

- Document 2: Stimulation results showing EPSP/IPSP
- Document 3: Neurotoxin effects on Ca**2⁺, GABA release

**Animation Scenes:**
1. **Scene 1a** - Draw neuron circuit (N1 -> M, N2 -> M)
2. **Scene 1b** - Show EPSP from N1, IPSP from N2 (identify synapse types)
3. **Scene 1c** - Calculate minimal stimulations needed for action potential
4. **Scene 2** - Explain M's integrator role (summation)
5. **Scene 3a** - Toxin T1 effect: blocks Ca**2⁺ entry  ->  no GABA release
6. **Scene 3b** - Toxin T2 effect: blocks GABA release directly
7. **Scene 3c** - Toxin T3 effect: blocks GABA receptors
8. **Scene 3d** - Complete mechanism: Ca**2⁺  ->  vesicle fusion  ->  GABA  ->  receptor

**Duration:** ~4 minutes

## 🎨 Animation Style Guidelines

### Visual Elements:
- **Graphs:** Use exact data points from exercises
- **Tables:** Recreate with actual values
- **Diagrams:** Match exercise figures
- **Colors:** Consistent coding (blue=given, red=calculated, green=answer)

### Text Elements:
- **Problem text:** Show verbatim from exercise
- **Steps:** Number each calculation step
- **Formulas:** Use LaTeX for all equations
- **Units:** Always include and highlight

### Timing:
- **Problem setup:** 10-15 seconds
- **Each step:** 5-10 seconds
- **Answer highlight:** 5 seconds
- **Transitions:** 1-2 seconds
